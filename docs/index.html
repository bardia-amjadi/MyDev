سلام :)

این یه پروژه‌ی شخصیه برای تمرین . نه شرکت پشتشه، نه تیم آنچنانی. فقط خودمم، دارم چیزایی که یاد می‌گیرم رو اینجا می‌چینم.

اگه اومدی دنبال دوره یا خدمات خاص، راستش اینجا چیز خاصی نیست. ولی شاید بین کدها یه چیزی پیدا کنی که به کارت بیاد.

داستانش چیه؟

من چند ساله دارم برنامه‌نویسی می‌کنم و تصمیم گرفتم یه پروژه ساده بزنم تا هم یادم بمونه چی یاد گرفتم، هم اگه کسی خواست نگاه کنه. این سایت یه جور دفترچه تمرینه برای من.

چی استفاده شده؟

  • Docker برای بالا آوردن محیط پروژه
  • Asp.Net Core Web API سمت بک‌اند
  • React.js برای فرانت
  • SQL Server برای دیتابیس

+ یه ذره HTML و Bootstrap که معلومه :)
